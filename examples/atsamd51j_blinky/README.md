# `atsamd51j_blinky`

An RTIC blinky example intended for the [pygamer].

[paygamer]: https://www.adafruit.com/product/4242

## Dependencies

#### 1. `hf2`:

```console
$ cargo install cargo-hf2
```

## Run

The [pygamer] needs to be connected to the computer via USB.

Then, run:

```
cargo hf2 --release
```
